# Real Estate RAG Tool

A Retrieval-Augmented Generation (RAG) application built with:

- LangChain
- ChromaDB
- HuggingFace Embeddings
- Groq LLM
- Streamlit

## Features

- Load real estate news URLs
- Convert them into embeddings
- Store in Chroma vector DB
- Ask contextual questions
- Get answers with sources

## Tech Stack

- Python
- LangChain
- Sentence Transformers
- Streamlit
- ChromaDB

---

This is a learning project built to understand RAG architecture end-to-end.
